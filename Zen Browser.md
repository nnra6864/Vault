## about:config

- `toolkit.legacyUserProfileCustomizations.stylesheets`: true - enables chrome dir theming
- `zen.theme.content-element-separation`: `0` - removes gaps around the browser
- `zen.theme.hide-tab-throbber`: `false` - enables the tab loading animation in the sidebar
- `zen.view.sidebar-collapsed.hide-mute-button`: `false` - doesn't hide the mute button in collapsed mode
- `zen.urlbar.show-contextual-id`: `true` - shows tab's container in the url bar
- `zen.view.grey-out-inactive-windows`: `false` - don't gray out inactive windows even more
- `zen.urlbar.replace-newtab`: `false` - enables the `Home` section in settings so you can replace the home page etc.
