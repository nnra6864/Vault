### [Jonas Eriksson](https://www.erikssonjonas.com/)

[Tape Fiasco](https://www.erikssonjonas.com/tapefiasco) - Tape Fiasco is a creative audio effect plugin that combines three distinct time-based effects: Stretch (granular time-stretching), Varispeed (tape-style speed manipulation), and Stutter (rhythmic slicing and repetition). Each effect can be used independently or in combination, with configurable signal routing between them. The plugin is designed for real-time performance and sound design, offering tempo-synced operation, envelope follower modulation, and extensive parameter control for each effect section.
